Universal Configurator V3.3.0 Update

INSTALL
1. Back up and commit V3.2.0.
2. Copy this update over the project.
3. Do not replace existing dealer settings or live data/freight-rules.csv.
4. To test freight, copy templates/freight-rules.csv to data/freight-rules.csv only if no live file exists.
5. Run through VS Code Live Server.

V3.3.0 adds the Universal Freight Engine, Freight inspector tab, expanded rule statistics, performance history summaries, and a diagnostics health report.

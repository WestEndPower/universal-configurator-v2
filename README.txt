Universal Configurator V3.2.0 Update

INSTALL
1. Back up and commit V3.1.0.
2. Copy this update over the project.
3. Do not replace existing dealer settings or live data/promotions.csv.
4. To test promotions, copy templates/promotions.csv to data/promotions.csv only if no live file exists.
5. Run through VS Code Live Server.

V3.2.0 adds a Universal Promotion Engine, Promotions inspector tab, shared pricing adjustments, and diagnostics export integration.

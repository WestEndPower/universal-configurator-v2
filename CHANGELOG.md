# Universal Configurator V3.2.0

- Added data-driven Universal Promotion Engine.
- Added optional `data/promotions.csv` loader.
- Added Dollar Rebate, Percent Discount, and Fixed Price promotion types.
- Added date, brand, product, SKU, category, and system applicability.
- Added priority, stackability, financing requirements, caps, customer text, and dealer funding.
- Added Promotions tab to CPQ Inspector.
- Added promotion evaluation to Rule Trace and diagnostics export.
- Promotion savings now flow through the shared configuration totals.
- Live promotion data is not included or overwritten; a starter template is included.
